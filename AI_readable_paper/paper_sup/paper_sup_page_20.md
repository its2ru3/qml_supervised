**FIG. 8.** Classification results for the quantum variational method for Sets I (a), II (b), and III (c) as a function of circuit depth. For each set and each depth a total of 20 test sets consistings of 20 data points per lable are randomly drawn and classified. The black dots are the mean value of the classification successes for each set at each depth, with the standard error of the mean given by the error bars. Each depth also contains a density plot showing the histograms for all 20 test sets classified. The width of each histogram bin corresponds to the precision of our success gradation and is equal to $1/40 = 0.025$. The combined results for the three plots are shown in the main text (Fig. 2c). The red dashed lines show the average success results for each set of the direct kernel method, for comparison.
